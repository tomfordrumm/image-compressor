Необходимо написать сжиматель изображений используя библиотеку browser-image-compression 

Проект должен быть написан на Vue. 

## UI/UX

-Зона загрузки:
Drag And Drop зона куда пользователь может закинуть файл либо при клике отркоет диалоговое окно где он сможет выбрать файл

-Панель настроек:
Input maxSizeMB для ограничения итогового веса файла
Input maxWidthOrHeight для ограничения итоговой ширны или высоты

-Кнопка оптимизировать

-Блок предпросмотра:
Слева картинка до сжатия
Справа картинка после сжатия

-Информационный блок:
Размер до
Размер после 
Экономия (процент насколько размер после стал меньше чем до)

-Кнопка скачать:
Пока идет процесс сжатия вместо кнопки должен быть спиннер


## Важно
приложение должно работать в static моде
